# VioletStream Standalone

Ce dossier contient une version simplifiée et autonome de VioletStream. Elle est conçue pour être hébergée facilement sur n'importe quel service de fichiers statiques (GitHub Pages, Netlify, Vercel, ou même un simple serveur FTP).

## Fichiers inclus :
- `index.html` : La structure principale du site.
- `style.css` : Les styles personnalisés et les animations.
- `app.js` : La logique du site (React).
- `data.js` : Les données des films (titres, descriptions, liens).

## Comment l'utiliser :
1. Téléchargez ou copiez le contenu de ce dossier `/standalone`.
2. Modifiez le fichier `data.js` pour ajouter vos propres films et liens.
3. Modifiez le lien Discord dans `app.js` (recherchez `https://discord.gg/...`).
4. Téléversez tous les fichiers sur votre hébergeur.

## Avantages :
- **Aucune base de données requise** : Tout est géré dans le navigateur.
- **Aucun serveur requis** : Fonctionne sur n'importe quel hébergeur statique.
- **Facile à modifier** : Les fichiers sont lisibles et simples.
